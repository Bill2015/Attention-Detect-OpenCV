
# Introduction (介紹)
## Attention Detect (分心檢測)
> Using Python with package opencv and dlib to detect the head rotation. And judge the user are pay attention on online learning.   
>> 利用 Python 與套件 OpenCV 和 Dlib 偵測頭部轉向，判斷使用者是否在線上學習是否有分心
